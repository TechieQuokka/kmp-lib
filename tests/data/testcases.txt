# KMP String Matching Library Test Cases
# Format: TEXT|PATTERN|EXPECTED_POSITIONS (comma-separated) or NOT_FOUND
# Lines starting with # are comments

# =============================================================================
# SECTION 1: Basic Matching Tests
# =============================================================================

# Simple exact match
hello world|world|6
The quick brown fox jumps over the lazy dog|fox|16
Programming in C++ is both challenging and rewarding|C++|15

# Match at beginning
The beginning of wisdom is the definition of terms|The|0
ABCDEFGHIJKLMNOPQRSTUVWXYZ|ABC|0
Start of the string matters for performance|Start|0

# Match at end
This is the final word|word|18
Looking for the needle in this haystack needle|needle|16,40
All roads lead to Rome|Rome|18

# Match in middle
The beautiful sunset painted the sky with brilliant colors|sky|33
In the middle of difficulty lies opportunity|middle|7
Between two evils I always pick the one I never tried before|evils|12

# Multiple occurrences
abracadabra|abra|0,7
Mississippi|issi|1,4
banana banana banana|banana|0,7,14
the the the the|the|0,4,8,12
aaaaaaaaa|aaa|0,1,2,3,4,5,6

# No match found
hello world|xyz|NOT_FOUND
The quick brown fox|elephant|NOT_FOUND
ABCDEFG|HIJKLMN|NOT_FOUND
completely different strings here|notfound|NOT_FOUND
